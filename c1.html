2. Structure of a C Program

A C program is divided into several sections for readability and execution:

  1. Documentation Section: Comments using // or /* */
  2. Preprocessor Section: Includes header files using #include
  3. Definition Section: Defines constants using #define
  4. Global Declaration: Variables and functions accessible throughout program
  5. main() Function: Starting point of execution
  6. Local Declaration: Variables inside functions
  7. Executable Section: Statements and logic
  8. User-defined Functions: Programmer-created functions
#include <stdio.h>
#define PI 3.14

int main() {
    int a = 5;
    printf("Hello BITM Students");
    return 0;
}

3. Tokens in C

  • Keywords: Reserved words like int, float, return
  • Identifiers: Names of variables/functions. Must start with letter/underscore and not be a keyword
  • Constants: Fixed values 10, 3.14, 'A'
  • Strings: Sequence of characters in quotes: "Hello"
  • Operators: Symbols for operations: +, -, *, /, >, &&
  • Delimiters: Symbols to separate tokens: , ; () { }

5. Identifier Rules

  • Must start with letter or underscore
  • Can contain letters, digits, underscore
  • Cannot be a keyword

6. Compilation Process in C

  • Preprocessing: Handles #include and #define
  • Compilation: Converts source code to assembly code
  • Assembly: Converts assembly code to machine code
  • Linking: Combines all files and libraries
  • Execution: Runs the program

2. Input two numbers and display sum

#include <stdio.h>
int main() {
    int a, b, sum;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}

3. Area of Rectangle

#include <stdio.h>
int main() {
    float length, width, area;
    printf("Enter length and width: ");
    scanf("%f %f", &length, &width);
    area = length * width;
    printf("Area = %.2f", area);
    return 0;
}

4. Celsius to Fahrenheit

#include <stdio.h>
int main() {
    float c, f;
    printf("Enter temperature in Celsius: ");
    scanf("%f", &c);
    f = (c * 9/5) + 32;
    printf("Temperature in Fahrenheit: %.2f", f);
    return 0;
}

5. Swap two numbers using third variable

#include <stdio.h>
int main() {
    int a, b, temp;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    temp = a;
    a = b;
    b = temp;
    printf("After swapping: a = %d, b = %d", a, b);
    return 0;
}
